üé® Colour & Contrast

Our colour palette is crafted to evoke a modern, gamer-friendly aesthetic that enhances visual engagement while supporting readability.

  • Primary Background (#0b1c2c): Deep navy tone reduces visual fatigue and supports light elements.
  • Secondary Background (#2c3e50): Provides visual grouping and sectioning without overwhelming the user.
  • Accent Color (#3498db): Used for buttons and highlights to guide attention and indicate interaction.
  • Text Color (#ecf0f1): High contrast and fully accessible, optimized for dark backgrounds.

This visual hierarchy improves attention flow and usability across all modules.

üß≠ Affordances

Interactive components are visually distinct through design metaphors, signaling what is clickable, expandable, or focusable.

  • Buttons include soft shadows and hover effects to simulate depth and responsiveness.
  • Accordion items use arrow indicators and background shifts to clearly communicate interactivity.
  • Form fields and search bars use rounded borders and padding to indicate input affordance.
  • Layout uses whitespace and grouping to intuitively organize actions and content blocks.

These affordances improve interaction efficiency and reduce user hesitation across devices.
